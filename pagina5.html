Con el dropdown logramos ahorrar espacio del menú ademas de estructurar mejor la informacion que estamos utilizando para el sitio web, en este caso, tendremos que crear un etiqueta "li" y aquí dentro hacer todo el trabajo, para el recuadro del desplegable, deberemos de hacer un div.

<ul class="w3-navbar w3-blue">
<li><a href="#">Link</a></li>
<li><a href="#">Link</a></li>
<li class="w3-dropdown-hover">
<a href="#">Dropdown</a>
<div class="w3-dropdown-content w3-white w3-card-4">
<a href="#">Link 1</a>
<a href="#">Link 2</a>
<a href="#">Link 3</a>
</div>
</li>
</ul>

Para lograr ese efecto, nos tenemos que meter en la etiqueta "a" de html y añadirle una class la cual nos permita que al pasar el cursor por encima de los links, estos cambien de color.

<ul class="w3-navbar w3-blue">
<li><a class="w3-hover-green" href="#">Link</a></li>
<li><a class="w3-hover-yellow" href="#">Link</a></li>
<li><a class="w3-hover-brown" href="#">Link</a></li>
<li><a class="w3-hover-red" href="#">Link</a></li>
</ul>
