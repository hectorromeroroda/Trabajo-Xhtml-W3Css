W3.CSS Cards

Este componente es un recuadro, que nos sirve de base para diseñar objetos mas complejos, como pueden ser: formularios, mensajes emergentes... o cualquier tipo de objeto que necesite como base un recuadro. Dispone de varias funcionalidades como: cambiar el color, la forma, el tamaño, efectos etc.

Card Content:

En este ejemplo se muestra el uso de w3-card, con un contenido estructurado con un header, contenido y footer.

Esto es el header

Aqui va el contenido que queramos poner.
(Ese texto esta alargado gracias a w3-wide)

Esto es el footer
Codigo de ejemplo:
<div class="w3-card-4 w3-border">
<header class="w3-container w3-green w3-text-shadow">
<h3>Esto es el header</h3>
</header>
<div class="w3-container w3-wide w3-bold">
<p>Aqui va el contenido que queramos poner. <br>(Ese texto esta alargado gracias a w3-wide)</p>
</div>
<footer class="w3-container w3-green w3-text-shadow">
<h5>Esto es el footer</h5>
</footer>
</div>
Photo Card:

En este ejemplo se muestra el uso de w3-card, con una imagen como contenido, un titulo como header, y el footer como explicacion.

Un ejemplo con una fotografia

Hasta el agua puede ser detenida por el frio

Codigo de ejemplo:
<div class="w3-card-4 w3-border w3-margin-left w3-margin-right">
<header class="w3-container w3-center w3-text-shadow">
<h4>Un ejemplo con una fotografia</h4>
</header>
<img src="imagenes/hielo.jpg" alt="Cascada de hielo" style="width:100%">
<div class="w3-container w3-center">
<p>Hasta el agua puede ser detenida por el frio</p>
</div>
</div>
